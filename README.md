# ✋ FingerBoard — рисование пальцами через камеру

Интерактивная доска для рисования, управляемая жестами руки в реальном времени.  
Никаких мышей и стилусов — только указательный палец и камера.

## 💡 Идея проекта

Приложение захватывает видео с веб-камеры, определяет положение кончика указательного пальца через MediaPipe Hands и использует его как «стилус» для рисования на виртуальном холсте. Холст накладывается поверх изображения с камеры (AR-режим). Пользователь может выбирать цвет, толщину линии и очищать холст, не касаясь клавиатуры — только жестами или касанием UI-панели пальцем.

## 🔁 Итерации

### Итерация 1 — базовые фичи (дедлайн: 26.04)

| # | Фича | Описание |
|---|------|----------|
| 1 | **Детекция указательного пальца** | Через MediaPipe Hands определяем landmark `INDEX_FINGER_TIP`; кончик пальца становится «курсором» |
| 2 | **Рисование на холсте** | Пока палец поднят и находится вне UI-зоны — рисуем линию по траектории. Прерываем линию, если палец исчез из кадра |
| 3 | **Выбор цвета** | Боковая панель с цветными кружками; наведение пальца + удержание 0.5 с меняет активный цвет |
| 4 | **Кнопка очистки холста** | Кнопка «Clear» на панели; при наведении пальца на неё холст сбрасывается |

### Итерация 2 — продвинутые фичи (дедлайн: TBD)

| # | Фича | Описание |
|---|------|----------|
| 1 | **Изменение толщины линии** | Жест «щипок» (расстояние между большим и указательным пальцем) плавно меняет `brush_size` от 2 до 30 px |
| 2 | **Ластик** | Жест «ладонь открыта» (все 5 пальцев подняты) переключает в режим ластика; повторный жест возвращает кисть |
| 3 | **Сохранение рисунка** | Жест «OK» (большой + указательный в кольцо) сохраняет холст в `output/drawing_<timestamp>.png` |

## ✅ Уточнения по использованию (для итерации 1)

- Рисование работает только вне левой UI-панели (зоны цветов и `Clear`).
- Для выбора цвета нужно удерживать указательный палец на цвете около `0.5` секунды.
- Для очистки холста нужно удерживать палец на кнопке `Clear` около `0.5` секунды.
- Если палец пропадает из кадра, текущая линия прерывается корректно и продолжается только при новом движении.
- Выход из приложения: `Q` или `Esc`.

## 📚 Используемые библиотеки вне курса

### MediaPipe Hands (`mediapipe`)

Библиотека от Google для детекции и трекинга руки в реальном времени.  
В проекте используется для определения положения кончика указательного пальца (`INDEX_FINGER_TIP`) в каждом кадре.
