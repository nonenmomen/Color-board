# ✋ FingerBoard — рисование пальцами через камеру

Интерактивная доска для рисования, управляемая жестами руки в реальном времени.  
Никаких мышей и стилусов — только указательный палец и камера.

## 💡 Идея проекта

Приложение захватывает видео с веб-камеры, определяет положение кончика указательного пальца через MediaPipe Hands и использует его как «стилус» для рисования на виртуальном холсте. Холст накладывается поверх изображения с камеры (AR-режим). Пользователь может выбирать цвет, толщину линии и очищать холст, не касаясь клавиатуры — только жестами или касанием UI-панели пальцем.

## 🚀 Как запустить

```bash
# 1. Клонируй репозиторий
git clone https://github.com/USERNAME/REPO_NAME.git
cd REPO_NAME

# 2. Создай и активируй виртуальное окружение
python -m venv .venv
source .venv/bin/activate      # Linux/macOS
.venv\Scripts\activate         # Windows

# 3. Установи зависимости
pip install -r requirements.txt

# 4. Запусти приложение
python main.py
```

**Требования:** Python 3.10+, веб-камера.
